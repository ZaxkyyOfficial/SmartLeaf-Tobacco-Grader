# SmartLeaf Tobacco Grader 🌿
**Inovasi Aplikasi Android untuk Sortasi Mutu Daun Tembakau Berbasis Klasifikasi On-Device**

---

## 🚀 Tentang Proyek
SmartLeaf Tobacco Grader adalah solusi digital agritech yang dirancang untuk menjawab tantangan standarisasi mutu daun tembakau secara objektif di tingkat petani dan pengepul. Dengan mengintegrasikan *Machine Learning (On-Device Inference)*, aplikasi ini mampu melakukan klasifikasi kematangan dan sortasi mutu daun secara *real-time* tanpa ketergantungan koneksi internet, menjamin efisiensi alur kerja di lapangan.

**SmartLeaf** dikembangkan dalam kerangka **Program Kreativitas Mahasiswa - Karsa Cipta (PKM-KC) 2026** di bawah naungan **Politeknik Elektronika Negeri Surabaya (PENS)**.

---

## ✨ Fitur Unggulan
* **On-Device Tobacco Grading:** Klasifikasi mutu berbasis AI yang berjalan lokal, memastikan privasi dan kinerja cepat di wilayah dengan konektivitas rendah.
* **Real-time Analytics:** Dasbor visual interaktif untuk memantau distribusi kematangan dan produktivitas grading harian.
* **Spatial Maturity Mapping:** Integrasi *geo-tagging* untuk pemetaan titik lokasi panen dengan akurasi tinggi.
* **Data Interoperability:** Fitur ekspor data ke format **CSV** yang kompatibel dengan sistem administrasi gudang dan Excel.
* **Professional UX:** Antarmuka berbasis *Material Design 3* yang dioptimalkan untuk penggunaan intensif di lapangan.

---

## 🛠 Teknologi yang Digunakan
* **Framework:** Android SDK (Kotlin) dengan Clean Architecture.
* **Architecture Pattern:** MVVM (Model-View-ViewModel) + Hilt (Dependency Injection).
* **Database:** Room Persistence Library (Offline-first approach).
* **AI/ML:** TensorFlow Lite (On-device model inference).
* **Charting:** MPAndroidChart (Data visualization).
* **API/Services:** Google FusedLocationProvider (Geo-tagging) & CameraX (High-performance vision).

## 📥 Panduan Instalasi
1.  Pastikan perangkat Anda menggunakan **Android API Level 24+**.
2.  Clone repository ini: `git clone https://github.com/username/smartleaf-tobacco-grader`
3.  Buka project melalui **Android Studio**.
4.  Sinkronkan Gradle dan pastikan `build.gradle.kts` sudah memuat semua *dependency* yang diperlukan.
5.  Jalankan aplikasi pada perangkat fisik untuk menguji fitur kamera dan GPS.
